База GeoLite2========

Ссылка на файл скачивания указан в конфиге = GeoLite2_IPv4_download

Для ручного обновления необходимо:
1. Открыть страницу скачивания = https://dev.maxmind.com/geoip/geoip2/geolite2/
2. Скачать файл = GeoLite2 City (binary, gzipped)
3. Распаковать архив в папку проекта = \files\GeoLite2\
4. Оставить в папке только файл GeoLite2-City.mmdb, а остальное содержимое архива удалить из папки
База обновлена!

Аналогично и по базе IPv6:
1. Открыть страницу скачивания = https://dev.maxmind.com/geoip/geoip2/geolite2/
2. Скачать файл = GeoLite2 City IPv6 (binary, gzipped)
3. Распаковать архив в папку проекта = \files\GeoLite2\
4. Оставить в папке только файл GeoLite2-City_ipv6.mmdb, а остальное содержимое архива удалить из папки


--------------------------------------------------

Поддержка IPv6========

Если появится геобаза GeoLite2 для IPv6, то нужно подключить ссылку на скачивание базы в конфиге, в параметре GeoLite2_IPv6_download (аналогично ссылке на скачивание IPv4 в параметре GeoLite2_IPv4_download)

После этого база будет задействована так же, как и IPv4 (т.е. база IPv6 будет обновляться при нажатии по кнопке "Обновить геобазу" и будет присваивать гео-информацию IPv6-адресам при их проверке)

--------------------------------------------------

Права на запись========

Открыть права на запись для папки files (755)
Чтобы обновлять геобазу.

--------------------------------------------------

Установка базы данных========

Открыть адрес /admin/main/install?password=12345, где 12345 - пароль админа из конфига
После этого будут созданы таблицы приложения


--------------------------------------------------

Включение капчи========

Включается в config/config.php
Запрашивать капчу у админа (параметр show_captcha_admin установить в true - да, false - нет)
Запрашивать капчу у пользователя (параметр show_captcha_user установить в true - да, false - нет)
